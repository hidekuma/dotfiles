---
name: explain
description: Generate visual explanation of code or architecture. Creates HTML presentation or ASCII diagrams.
---

Create a visual explanation of the specified code or system:

1. **Overview** - What is this? What problem does it solve?
2. **Architecture** - ASCII diagram showing components and data flow
3. **Key Concepts** - 3-5 most important things to understand
4. **Code Walkthrough** - Step-by-step explanation of critical paths
5. **Gotchas** - Non-obvious behaviors, edge cases, common mistakes

Format as either:
- ASCII diagrams for terminal viewing
- HTML presentation if requested (with embedded styles)

Keep explanations clear and visual. Use analogies where helpful.
